'House on Fire'
===============

##### Multiplayer Spiel für öffentliche Installationen
In diesem Projekt geht es darum ein Multiplayer Spiel für Handys zu entwickeln, wobei das Spiel auf einer großen Projektionsfläche angezeigt wird und das Handy lediglich als Gamecontroller dient um auf das Spiel Einfluss zu nehmen. So könnte zum Beispiel durch Bewegung des Handys, durch Berühren des Touchscreen oder durch Senden einer SMS das Spiel gesteuert werden. Das Spiel wird so aufgebaut sein, dass bestimmte Aufgaben nur gemeinsam gelöst werden können bzw. das mit zunehmenden Levels immer mehr Spieler benötigt werden um ein Level zu meistern.

##### Remember
Ensure objects transmitted over the network have the same package name
