## Reserviert den Platz für die benötigten Assets!

#### Vorraussetzungen für die Sound- und Musikdateien:

##### 1. Format:
+ wav (nicht empfohlen)
+ ogg 
+ mp3


[Convert mp3 and wav to ogg](http://audio.online-convert.com/convert-to-ogg)

##### 2. Size
Sounddateien können bis zu 1MB groß sein.

Musikdateien können eine beliebige Größe haben.


##### 3. Sampling
WAV files müssen 16-bit signed PCM sein.
